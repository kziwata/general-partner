# Partner Workspace

このリポジトリ（general-partner）の永続作業場。

## Files

| File | Purpose |
|------|---------|
| [INBOX.md](./INBOX.md) | 未整理の入力 |
| [TASKS.md](./TASKS.md) | 永続タスク |
| [DECISIONS.md](./DECISIONS.md) | 決定事項ログ |
| [NOTES.md](./NOTES.md) | 整理済み参考情報 |
| [sessions/](./sessions/) | セッション記録 |

## 他プロジェクトへ展開

新しいプロジェクトでは [docs/partner-template/](../docs/partner-template/) を `docs/partner/` にコピーして使う。

## 個人横断ハブ

プロジェクトをまたぐタスクは `~/.cursor/partner/` も併用する。
